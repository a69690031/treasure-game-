## Commands

```bash
npm install       # Install dependencies
npm run dev       # Start dev server at http://localhost:3000 (opens automatically)
npm run build     # Build to ./build directory
```

There is no test runner configured in this project.

## Architecture

This is a single-page React + TypeScript app built with Vite. All game logic lives in `src/App.tsx` — there are no routes or additional pages.

**Key files:**
- `src/App.tsx` — entire game: `Box` interface, `initializeGame`, `openBox`, and `resetGame` logic, plus all JSX
- `src/assets/` — chest images (`treasure_closed.png`, `treasure_opened.png`, `treasure_opened_skeleton.png`, `key.png`)
- `src/audios/` — sound effects (`chest_open.mp3`, `chest_open_with_evil_laugh.mp3`)
- `src/components/ui/` — full shadcn/ui component library (pre-generated, rarely needs editing)
- `src/components/figma/ImageWithFallback.tsx` — image component with graceful error fallback

**Styling:** Tailwind CSS v4 (compiled into `src/index.css` — do not edit this file directly). Use Tailwind utility classes in JSX. The `src/styles/globals.css` file is where custom global styles go.

**Path alias:** `@` maps to `src/` (configured in `vite.config.ts`).

**Animations:** Uses `motion/react` (Motion library) for chest open/hover animations.

**Game logic summary:** 3 treasure boxes are created on mount with one randomly assigned treasure. Clicking a box calls `openBox` which updates score (+100 for treasure, -50 for skeleton) and ends the game when treasure is found or all boxes are opened.
